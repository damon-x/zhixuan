# 需求澄清和疑问解答

## 前端技术栈
- **疑问**："前端用 nodejs + vue" 表述不清
- **解答**：开发阶段用 node 环境，nodejs 是开发阶段的基础环境

## 移动端适配
- **疑问**：移动端是响应式还是单独开发？
- **解答**：移动端网页适配，不是 app

## 数据库选型
- **疑问**：SQLite 到 MySQL 迁移会有差异
- **解答**：使用 GORM 屏蔽差异

## 登录状态
- **疑问**："不失效 cookie" 有安全风险
- **解答**：坚持使用"不失效 cookie"

## 对话记录存储
- **疑问**：每条对话一条记录会有性能问题
- **解答**：坚持原设计，每条对话一条记录

## AI 任务接口
- **疑问**：tool 不落库会有数据一致性风险
- **解答**：设计就是前端确认后才落库，没有一致性风险

## 其他方面
- **疑问**：错误处理、安全性、性能、监控等方面缺失
- **解答**：当前阶段不是考虑这些的时候
